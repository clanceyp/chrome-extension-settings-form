Getting started

This extension displays a queue of STASH pull requests you have been named in.

Setup

To display STASH requests, the first thing you need to do is login to your STASH instance and add your STASH URL to the options.

Once you have entered your STASH URL, check the popup display is working.

To keep a track of pull requests you initiated change the query string param in the "Stash rest" item.

Reset settings

Items with a default value can be reset by double clicking on the element.

To reset all the setting back to the default setting, click on the "Reset default options" button.
